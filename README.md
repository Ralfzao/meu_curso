# Calculadora em Python e Shell Script 

Neste repositório contem 2 arquivos da calculadora:
- Uma versão em Python (`Calculadora.py`)
- Uma versão em Shell Script (`calculadora.sh`)


---

## Requisitos

- Python 3 instalado
- Sistema operacional compatível com execução de scripts `.sh` (Linux, macOS ou WSL no Windows)

---

## Como executar o arquivo `.sh`

1. Acesse a pasta do projeto pelo terminal.
2. Caso necessário, conceda permissão de execução ao arquivo:

```bash
chmod +x calculadora.sh
```

---

## Como executar a calculadora em Python (`calc.py`)

1. Acesse a pasta do projeto pelo terminal do seu editor de código.
2. Execute o comando:

```bash
python calc.py
```

---

## Manuseio da calculadora

 - Escolha o primeiro número que deseja calcular.
 - Escolha a operação matemática desejada(+, -, /, *, **, p%).
 - Escolha o segundo número e aperte enter.
 - Caso a operação seja invalida aparecera uma aviso e precisará refazer sua operação.
 - No final poderá escolher entre continuar ou fechar a calculadora.
